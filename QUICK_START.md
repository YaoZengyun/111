# 快速开始指南

## 🚀 5分钟快速上手

### 步骤 1: 安装应用（1分钟）

1. 用 **Android Studio** 打开项目文件夹
2. 连接你的Android手机（或启动模拟器）
3. 点击绿色的运行按钮 ▶️
4. 等待应用安装到手机

### 步骤 2: 启用无障碍服务（2分钟）

应用首次打开时会自动显示引导对话框：

```
┌─────────────────────────────────┐
│  如何启用无障碍服务              │
├─────────────────────────────────┤
│  📱 设置步骤：                  │
│  1. 点击下方按钮打开无障碍设置   │
│  2. 找到并点击「QQ消息图片助手」│
│  3. 打开右上角的开关启用服务     │
│  4. 在弹出的对话框中点击「允许」│
│  5. 返回应用，确认服务状态       │
│                                 │
│  [去设置]  [稍后]              │
└─────────────────────────────────┘
```

**点击「去设置」→ 找到应用 → 打开开关 → 允许**

✅ 回到应用，看到"✓ 无障碍服务已启用"即成功

### 步骤 3: 选择模板图片（1分钟）

1. 在应用中点击 **「选择模板图片」**
2. 从相册选择一张图片作为背景
3. 图片会显示在预览区域

### 步骤 4: 调整设置（1分钟）

调整文字在图片上的位置和样式：

- **X坐标**: 左右位置（0-1000）
- **Y坐标**: 上下位置（0-1000）
- **文字大小**: 字体大小（推荐30-60）
- **最大宽度**: 文字换行宽度
- **文字颜色**: 点击按钮选择颜色

调整完毕后点击 **「保存设置」**

### 步骤 5: 启用功能并测试

1. 打开应用顶部的 **「启用自动处理」** 开关
2. 打开QQ，进入任意聊天
3. 输入一些文字（如"测试123"）
4. 点击发送
5. 应用会自动处理并保存图片

### 通用设置
```
X坐标: 50-100（左边距）
Y坐标: 100-200（上边距）

最大宽度: 500
文字颜色: 黑色

```

## 🔍 故障排除

### 问题 1: 服务状态显示"未启用"

**解决方法：**
1. 重新进入无障碍设置
2. 关闭服务后重新开启
3. 重启手机
4. 检查应用是否被系统限制后台

### 问题 2: 选择图片后没有预览

**解决方法：**
1. 检查存储权限是否授予
2. 重新选择图片
3. 选择更小的图片（<5MB）

### 问题 3: 在QQ中发消息没反应

**检查清单：**
- [ ] 无障碍服务是否已启用
- [ ] 应用内开关是否打开
- [ ] 是否已选择模板图片
- [ ] QQ版本是否过旧（建议更新到最新版）

### 问题 4: 文字位置不对

**调整方法：**
1. 打开应用，实时调整X/Y坐标
2. 保存设置
3. 重新在QQ中测试
4. 多次尝试找到最佳位置
